# Realism Nutrition

Overhauls the vanilla hunger system by introducing thirst, a realistic nutrition model with calories, macronutrients, and metabolism.

### 🌍 Русская версия

См. README.md для русской версии мода.

---

## ✨ Features

- Macronutrients (proteins, fats, and carbohydrates) and vitamins
- Calories instead of the vanilla hunger system
- Gradual calorie absorption through metabolism
- Thirst, which affects the metabolism
- Compatibility with some farming and cooking mods (e.g. Farmer’s Delight)

---

## 📦 Installation

1. Install the Fabric
2. Download the fabric api
3. Download the mod ([Modrinth](https://modrinth.com))
4. Place the mod file and fabric api into your `.minecraft/mods` folder
5. Launch the game

---

## ⚙️ Requirements & Dependencies

- Minecraft: **1.20.1**
- Mod Loader: **Fabric 0.18.3**
- Fabric API: **0.92.7**
- Java: **17**
