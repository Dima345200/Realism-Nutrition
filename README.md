# Realism Nutrition

Перерабатывает ванильную систему сытости, добавляя жажду, реалистичную модель питания с калориями, макронутриентами и метаболизмом.

### 🌍 English version

See [README_EN.md](README_EN.md) for the English version of this mod.

---

## ✨ Особенности

- Макронутриенты (белки, жиры и углеводы) и витамины
- Калории вместо сытости
- Постепенное восполнение калорий после еды (метаболизм)
- Жажда, влияющая на обмен веществ
- Совместимость с некоторыми модами на фермерство и готовку (например, Farmer's Delight)

---

## 📦 Установка

1. Установите Fabric
2. Скачайте fabric api
3. Скачайте мод ([Modrinth](modrinth.com))
4. Переместите файл мода и fabric api в папку `.minecraft/mods`
5. Запустите игру

---

## ⚙️ Требования и зависимости

- Minecraft: **1.20.1**
- Mod Loader: **Fabric 0.18.3**
- Fabric API: **0.92.7**
- Java: **17**
