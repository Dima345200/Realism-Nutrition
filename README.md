# Realism Nutrition

Перерабатывает ванильную систему сытости, добавляя реалистичную модель питания с калориями, макронутриентами и метаболизмом.

---

## 🌿 Ветки и версии

Этот мод имеет разные ветки под разные версии Minecraft:

- `main` → последняя версия мода

📌 См. требования в README соответствующей ветки.

---

## ✨ Особенности

- Макронутриенты (белки, жиры и углеводы) и витамины
- Калории вместо сытости
- Постепенное восполнение калорий после еды (метаболизм)
- Механика жажды
- Совместимость с популярными модами на фермерство и готовку (Farmer's Delight)

---

## 📦 Установка

1. Установите нужный загрузчик (Fabric, Forge, Neoforge)
2. Скачайте мод под нужную версию Minecraft
3. Установите необходимые зависимости (если требуются)
4. Переместите файл мода в папку `.minecraft/mods`
5. Запустите игру
